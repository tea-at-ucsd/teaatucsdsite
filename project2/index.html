Project Overview

Using Raspberry Pi technology and lightweight machine learning models, this project introduces intuitive computer vision gesture-based interactions for attractions. Guests control elements of the experience with simple hand motions, enhancing immersion and creating an unforgettable Star Wars-inspired journey.

Key Features

  • ⚫ Real-time Computer Vision gesture detection
  • ⚫ Highly optimized machine learning model
  • ⚫ Animatronic features
  • ⚫ Seamless themed integration for maximum immersion
  • ⚫ Engaging storytelling with live actors
